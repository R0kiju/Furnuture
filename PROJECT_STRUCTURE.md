# Структура файлов проекта (Custom Code)

В этом списке перечислены только те файлы, которые относятся непосредственно к логике и дизайну сайта, исключая конфигурационные файлы фреймворков и библиотек.

## 🖥 Бэкенд (Сервер)
- `server/index.ts` — Основная логика сервера, маршруты API для товаров и заказов.
- `server/db.ts` — Логика взаимодействия с базой данных SQLite.
- `server/test.ts` — Скрипт для тестирования базы данных или API.
- `database.sqlite` — Файл самой базы данных.

## 🌐 Фронтенд (Сайт)

### Основные файлы
- `src/main.tsx` — Точка входа в React-приложение.
- `src/App.tsx` — Главный компонент, настройка маршрутов (Router).
- `src/index.css` — Глобальные стили проекта.
- `src/App.css` — Стили для основного макета.
- `public/favicon.svg` — Иконка сайта (Favicon).
- `public/icons.svg` — Набор SVG-иконок, используемых на сайте.

### Страницы (`src/pages/`)
- `Home.tsx` — Главная страница со списком товаров.
- `Checkout.tsx` — Страница оформления заказа и корзина.
- `AdminPanel.tsx` — Панель управления товарами и заказами.

### Компоненты (`src/components/`)
- `Header.tsx` — Навигационная панель и логотип.
- `ProductCard.tsx` — Карточка отдельного товара с кнопкой покупки.
- `Modal.tsx` — Универсальный компонент модального окна.

### Логика и состояние
- `src/store/cartStore.ts` — Управление состоянием корзины (добавление, удаление, подсчет суммы).
- `src/context/CartContext.tsx` — Дополнительный контекст (если используется в связке со стором).
- `src/hooks/useInView.ts` — Хук для анимации появления элементов при скролле.
- `src/types/index.ts` — Типы данных для товаров, заказов и пользователей.

### Данные и ассеты
- `src/data/products.ts` — Список товаров (если не загружается из БД).
- `src/assets/hero.png` — Главное изображение для баннера.
