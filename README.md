# Furnuture — Магазин мебели

Современное веб-приложение для продажи мебели, построенное на стеке React и Express с использованием SQLite.

## 🚀 Технологии

### Фронтенд (Frontend)
- **React 19**: Библиотека для создания пользовательских интерфейсов.
- **Vite**: Инструмент сборки, обеспечивающий быструю разработку.
- **TypeScript**: Типизированное надмножество JavaScript для повышения надежности кода.
- **Zustand**: Легковесное решение для управления состоянием (State Management).
- **React Router 7**: Навигация и маршрутизация в приложении.
- **Lucide React**: Набор иконок для интерфейса.
- **Vanilla CSS**: Стилизация выполнена на чистом CSS для максимальной производительности.

### Бэкенд (Backend)
- **Node.js**: Среда выполнения JavaScript на сервере.
- **Express 5**: Веб-фреймворк для создания API.
- **Better-sqlite3**: Быстрая библиотека для работы с базой данных SQLite.

### База данных
- **SQLite**: Легкая и надежная реляционная база данных, хранящаяся в файле `database.sqlite`.

## 🛠 Архитектура проекта

### Серверная часть (`/server`)
- `index.ts` — Основной файл сервера, обработка API запросов.
- `db.ts` — Настройка подключения к базе данных и инициализация схем.

### Клиентская часть (`/src`)
- `pages/` — Основные экраны приложения (Главная, Оформление заказа, Админ-панель).
- `components/` — Переиспользуемые UI-компоненты (Шапка сайта, Карточка товара, Модальные окна).
- `store/` — Глобальное состояние приложения (Корзина покупок).
- `hooks/` — Кастомные React-хуки для логики (например, отслеживание видимости элементов).
- `data/` — Статические данные или конфигурации.
- `types/` — Описания типов TypeScript для всего проекта.

## 🚦 Как запустить проект

1. **Установка зависимостей**:
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки**:
   ```bash
   npm run dev
   ```
   *Эта команда одновременно запускает и Vite (фронтенд) и Express (бэкенд).*

3. **Сборка проекта**:
   ```bash
   npm run build
   ```
